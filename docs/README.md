## 🎯 核心功能

### 支持的数据集

- ✅ **COCO** - 通过 `prepare_data.py` 或 `prepare_coco_from_download.py`
- ✅ **CUB-200-2011** - 通过 `prepare_cub_from_kaggle.py` 或 `download_cub.py`
- ✅ **自定义数据集** - 通过 `prepare_data.py`

### 主要脚本

- `train.py` - 训练 DiT 模型
- `inference.py` - 使用训练好的模型生成图像
- `prepare_data.py` - 准备 COCO 或自定义数据集
- `prepare_coco_from_download.py` - 处理手动下载的 COCO 数据
- `prepare_cub_from_kaggle.py` - 处理从 Kaggle 下载的 CUB 数据
- `download_cub.py` - 下载 CUB 数据集
